    ### 1-주기 신호

    `정현파와 비정현파로 분리`  
    정보를 변환하는 신호는 일정한 주기를 가지고 있음

    ```
    주기 신호 식
    x(t) = A(T=t)
    T : 주기
    ```

    ```
    정현파 식
    x(t) = Asin(wt+q)
    A : 진폭, w : 주파수 q : 위상
    ```

    진폭(A) : 신호의 크기나 세기를 의미, 단위:[V]볼트
